Import Application Terms

The BEDES Manager can import a csv file with implementation terms:

  • First line should be the header
  • Each additional line is an implementation term, either "Atomic" or "Constrained List"
  • Each of implementation term should have the following required columns:
    1. Term Name - The name of the application term, value is required.
    2. Term Description - The description of the application term.
    3. Term Type* - "[value]" or "Constrained List", value is required.
    4. Unit - Unit Text
  • Constrained List terms should define repeating pairs of List Option Name/Description columns for each constrained list option.
    1. List Option Name
    2. List Option Description
